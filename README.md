# DevJobs API REST 
Desenvolvida na jornada .NET Direto ao Ponto. API REST Completa de Gerenciamento e Aplicação de Vagas de Emprego
##### Tecnologias Utilizadas : 
- ASP.NET Core com .NET 6 
-  Entity Framework Core 
- SQL Server - Swagger 
- Logs com Serilogs 
- Injeção de Dependência
